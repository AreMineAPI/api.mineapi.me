## `üíª` Get Started
- Make sure your Node.js version is v18.12.1.
- Open your Terminal in the file directory and type the lines below in order.

  - ```bash
    npm install --save
    # or
    yarn install
    ```
  - ```bash
    npm run start
    # or
    yarn start
    ```
- ### How to run on `Development` server?
  - ```bash
    npm run dev
    # or
    yarn dev
    ```
