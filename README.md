## `üíª` Get Started
- Fill in the `src/config.js` file completely.
- Make sure your Node.js version is v16.16.0.
- Open your Terminal in the file directory and type the lines below in order.

  - ```bash
    npm install --save
    # or
    yarn install
    ```
  - ```bash
    npm run start
    # or
    yarn start
    ```
- ### How to run on `Development` server?
  - ```bash
    npm run dev
    # or
    yarn dev
    ```
